# ResilienceHub - User Guide for auto deploy

## Application Overview

ResilienceHub is a comprehensive web application that provides cognitive behavioral therapy interactive tools connecting mental health professionals with clients. The platform facilitates structured tracking, reflection, and progress monitoring without positioning itself as therapy.

## User Roles

### Client/Patient Role

**As a client, you can:**

1. **Track Emotions** 
   - Select emotions using the interactive emotion wheel
   - Record intensity and situational context
   - View emotion history and patterns over time

2. **Create Thought Records**
   - Document automatic thoughts and situations
   - Identify cognitive distortions (unhelpful thinking patterns)
   - Practice cognitive reframing with the Reframe Coach tool

3. **Journal Entries**
   - Write reflective journal entries
   - Receive optional AI-assisted analysis of emotions and themes
   - Track journaling progress and patterns

4. **Practice Cognitive Reframing**
   - Complete interactive practice sessions
   - Select helpful alternative thoughts
   - Receive immediate feedback and scoring
   - View practice history and improvement metrics

5. **Set Goals and Track Progress**
   - Create personal growth goals with specific milestones
   - Monitor progress through the interactive dashboard
   - Receive achievement badges for consistent progress

6. **View Cross-Component Insights**
   - See how emotions, thoughts, and journaling connect
   - Identify patterns across different tracking tools
   - Monitor protective factors and coping strategies usage

### Professional/Therapist Role

**As a professional, you can:**

1. **Manage Client Relationships**
   - Invite and manage client connections
   - View a comprehensive list of your clients
   - Access client profiles and history

2. **Review Client Records**
   - View clients' emotion tracking data
   - Review thought records and identify patterns
   - Read journal entries and provide comments

3. **Monitor Client Progress**
   - Track client engagement with the platform
   - View practice session results and accuracy
   - Identify cognitive distortion patterns

4. **Provide Guidance**
   - Share resources with specific clients
   - Comment on journal entries
   - Recommend specific practice areas

5. **Access Clinical Insights**
   - View cross-component data analytics
   - Identify trends across multiple clients
   - Track the effectiveness of different approaches

6. **Manage Professional Resources**
   - Create and organize educational resources
   - Share targeted resources with specific clients
   - Track resource usage and effectiveness

## Key Features

### Emotion Tracking
The application provides an interactive emotion wheel that helps users identify and record emotions with precision. The wheel is responsive, offering different interfaces for mobile and desktop users.

### Reframe Coach
The Reframe Coach feature provides interactive practice for cognitive reframing skills. Users are presented with scenarios containing cognitive distortions and must select the most helpful reframing option. The system tracks progress, provides scores, and maintains a detailed practice history.

### Dashboard Analytics
The dashboard presents personalized insights by analyzing data across different components of the application. It shows connections between emotions, thought patterns, and journal entries, helping both clients and professionals identify important patterns.

### Cross-Component Insights
This feature visualizes relationships between different data components, showing connections between emotion records, thought records, and journal entries. It helps users identify patterns in their mental health data across different tracking tools.

## Data Security and Privacy

- All data is securely stored and encrypted
- Client-professional relationships are strictly managed through the invitation system
- Professionals can only access data for their connected clients
- The platform uses secure authentication with role-based access controls
